Модуль коррекции ошибок Whisper в уже готовой транскрипции через словарь
`data/hotwords/combined_vocab.txt` (45K–86K терминов ВОВ/ВМВ).

**Цель:** исправить специфичные ошибки распознавания:

## 🧩 4 слоя коррекции

### Слой 1: Exact Match (case-insensitive)

**Алгоритм:**
- Токенизация текста (сохраняя пунктуацию)
- Прямое сравнение с vocab: `word.lower() == term.lower()`
- Замена с сохранением регистра из vocab

**Когда срабатывает:**
- Whisper правильно распознал слово, но с неверным регистром
- Пример: "жуков" → "Жуков", "приладожье" → "Приладожье"

---

### Слой 2: Fuzzy Match (Levenshtein distance ≤ 2)

**Алгоритм:**
- Для каждого слова: `distance(word, vocab_term) ≤ 2` → кандидат
- Если кандидатов >1 → приоритет по весу (weight DESC)
- Только для слов ≥4 символа

**Когда срабатывает:**
- Whisper ошибся на 1–2 символа
- Примеры:
  - "Симоняка" → "Симоняк" (distance=1)
  - "имплаздарм" → "плацдарм" (distance=2)
  - "Фукивульф" → "Фокке-Вульф" (distance=2)

---

### Слой 3: Phonetic Match (русский soundex)

**Алгоритм:**
- Фонетический хэш (упрощённый metaphone для русского):
  - Гласные опускаются (кроме первой)
  - Звонкие → глухие (б→п, д→т, г→к, в→ф)
  - Ь, Ъ опускаются
- Применяется только для фамилий (category=1,2 в vocab)

**Когда срабатывает:**
- Whisper перепутал похожие звуки
- Пример: "Коварова" → "Говоров" (К/Г похожи)

---

### Слой 4: Context-Aware (биграммы)

**Алгоритм:**
- Проверка окна ±2 слова:
  - Титул + фамилия: "генерал Коварова" → "генерал Говоров"
  - Биграммы: "1-й Белорусский фронт" (приоритет над одиночным "Белорусский")
- Только для категорий 1, 2, 5 (командиры, операции, воинские части)

**Когда срабатывает:**
- Vocab содержит устойчивое словосочетание
- Пример: "дивизии Симоняка" → context подсказывает род.п.

---

## ⚖️ Weight-Based Prioritization

Если найдено несколько кандидатов (Fuzzy/Phonetic):
1. Сортировка по `weight` (DESC)
2. При равных весах → по `distance` (ASC)
3. При равных distance → по длине термина (DESC)

**Пример:**
Слово: "Фукивульф"
Кандидаты:

"Фокке-Вульф" (вес=5, dist=2)

"вульф" (вес=1, dist=6)
→ Выбор: "Фокке-Вульф"

## 📋 15 примеров ошибок Whisper

### 1. Грамматика
**Вход:** "впечатляющаясь начать"  
**Ожидаемый выход:** "впечатляющей начать"  
**Слой:** Fuzzy (distance=3 с учётом окончания)

### 2. Техника (первое вхождение)
**Вход:** "Не были равны новейшим «Фукивульфам-190»"  
**Ожидаемый выход:** "«Фокке-Вульфам-190»"  
**Слой:** Fuzzy (distance=2, weight=5)

### 3. Техника (второе вхождение)
**Вход:** "располагая именно под Ленинградом двумя группами новейших четырехпушечных истребителей «Фукивульф-190»"  
**Ожидаемый выход:** "«Фокке-Вульф-190»"  
**Слой:** Fuzzy

### 4. Техника (третье вхождение)
**Вход:** "Самолеты с четырьмя пушками «Фукивульфа» могли расправляться"  
**Ожидаемый выход:** "«Фокке-Вульфа»"  
**Слой:** Fuzzy (сохранение падежа "-а")

### 5. Техника + аббревиатура
**Вход:** "Фактически «Фукивульф-190» был опасен и четырехмоторной летающей крепости, и штурмовику Л-2, и он мог догнать и советские бомбардировщики «Педва»."  
**Ожидаемый выход:** "«Фокке-Вульф-190»" + "«Пе-2»"  
**Слой:** Fuzzy для обоих

### 6. Название операции
**Вход:** "под готовительный период «Искры»"  
**Ожидаемый выход:** "«Искры»" (уже правильно, проверка regression)  
**Слой:** Exact (подтверждение)

### 7. Техника в контексте
**Вход:** "В результате четвёрки немецких фоке-вольфов"  
**Ожидаемый выход:** "фоке-вольфов" (уже правильно через дефис)  
**Слой:** Exact (подтверждение) или оставить без изменений

### 8. Топоним (регистр)
**Вход:** "была отрезана в приладожье"  
**Ожидаемый выход:** "Приладожье"  
**Слой:** Exact (case fix)

### 9. Фамилия (первое упоминание)
**Вход:** "В чем секрет успеха 136-й стрелковой дивизии «Симоняка»"  
**Ожидаемый выход:** "«Симоняка»" → **SKIP** (род.п., vocab содержит "Симоняк" им.п.)  
**Слой:** Context должен детектировать род.п., но в v1 оставляем как есть

### 10. Фамилия (второе упоминание)
**Вход:** "«Симоняк» был легендой Ленинграда"  
**Ожидаемый выход:** "Симоняк" (уже правильно)  
**Слой:** Exact (подтверждение)

### 11. Фамилия в списке
**Вход:** "которые были в первой линии в 67-й армии Ленинградского фронта успешно захватили плацдарм только две семняка и соседняя 268-я."  
**Ожидаемый выход:** "семняка" → "Симоняка" (?)  
**Слой:** Fuzzy (distance=2)  
**Проблема:** "семняка" может быть опечаткой Whisper для "Симоняка", но distance=4 → SKIP в v1

### 12. Фамилия (фонетическая)
**Вход:** "Идея Коварова о том, чтобы бить"  
**Ожидаемый выход:** "Говорова"  
**Слой:** Phonetic (К→Г) + Context (род.п. "-а")

### 13. Топонимы
**Вход:** "26 рухнула не только на западном Водолочного Горла, но и на восточном фасе, там, где наступал Волковский фронт"

### 14. Военный термин
**Вход:** "86-я дивизия становится вторым эшелоном для Симоника, ее переправляют на занятый имплаздарм"  
**Ожидаемый выход:** "имплаздарм" → "плацдарм"  
**Слой:** Fuzzy (distance=2)

### 15. Топоним (второе упоминание)
